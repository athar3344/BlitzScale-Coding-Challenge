OrderFetcher - a class that fetches orders.
OrderProcessor - a class that processes the orders.
OrderWriter - a class that writes the output to a file.
*********************************************************************

OrderFetcher Class
Overview
The OrderFetcher class is responsible for fetching the orders from a remote API URL and returning
 them as an array of orders.

*********************************************************************
OrderProcessor Class
Overview
The OrderProcessor class is responsible for processing the orders and generating output for each order.

*********************************************************************

OrderWriter Class
Overview
The OrderWriter class is responsible for writing the output of the orders to a CSV file and for returning 
the CSV file as a download to the user.
